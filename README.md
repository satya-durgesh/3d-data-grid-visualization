## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd "web design"
```

2. Install dependencies:
```bash
npm install
```

## Running Locally

Start the server:
```bash
npm start
```

The application will be available at `http://localhost:3000`

For development with auto-reload:
```bash
npm run dev
```

## Deployment

This project can be deployed to:
- **Heroku**: Uses the included `package.json` with start script
- **Vercel**: Configure as Node.js project
- **Netlify**: Add build command and publish directory
- **GitHub Pages**: Use static hosting (modify server.js or use static build)

### GitHub Pages Deployment

For static GitHub Pages deployment, you can serve the files directly without the Node.js server.
